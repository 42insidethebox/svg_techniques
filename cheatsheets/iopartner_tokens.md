# Visual Tokens

These are the default visual tokens used across the repository.

## Colors

```css
--io-blue: #2563eb;
--io-navy: #0f172a;
--io-dark: #061120;
--io-panel: #f8fbff;
--io-muted: #64748b;
--io-cyan: #93c5fd;
```

## Geometry

```css
--io-grid-size: 32px;
--io-panel-radius: 28px;
--io-node-radius: 16px;
--io-icon-stroke: 1.75;
--io-route-opacity: 0.28;
--io-grid-opacity: 0.055;
```

## Usage

Use these tokens as defaults.

Project-specific visuals may adjust them, but should preserve the same structure:

- one main accent color
- one dark color
- one panel color
- one muted text color
- consistent radius
- consistent stroke logic
