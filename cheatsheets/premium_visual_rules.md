# Premium Visual Rules

## The formula

```txt
subtle grid
+ low-opacity SVG curves
+ small technical marks
+ restrained shadows
+ consistent radius
+ one accent color
```

## Do

- Use fewer elements.
- Use thin strokes.
- Use low opacity.
- Use one accent color.
- Use consistent radius.
- Use whitespace generously.
- Align everything.
- Use the same icon style everywhere.

## Avoid

- Thick neon lines.
- Random colors.
- Random icon styles.
- Too many labels.
- Fake complexity.
- Heavy shadows everywhere.
- Decorative elements that fight the content.

## Quick diagnosis

If a visual looks cheap, reduce:

- opacity
- glow
- number of elements
- color count
- stroke width

If a visual looks empty, improve:

- alignment
- hierarchy
- spacing
- route rhythm
- node placement
